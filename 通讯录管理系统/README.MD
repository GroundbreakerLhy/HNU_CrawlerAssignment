## 如何使用

我的电脑装的是conda所以使用的是conda管理虚拟环境，源是清华源

首先创建虚拟环境

```powershell
conda creat -n data_venv python=3.10
```

然后安装依赖

```powershell
conda activate data_venv
pip install -r requirements.txt
```

接着你运行就行了
